##########################################
Susceptibility Distortion Correction (SDC)
##########################################

Please note that all routines for susceptibility-derived distortion correction
have been used in
`fMRIPrep <https://www.nipreps.org/fmriprep>`_,
`QSIPrep <https://github.com/PennBBL/qsiprep>`_,
`dMRIPrep <https://www.nipreps.org/dmriprep>`__, and other projects.
For more detailed documentation on
:abbr:`SDC (susceptibility-derived distortion correction)` routines,
check `www.nipreps.org/sdcflows <https://www.nipreps.org/sdcflows>`_.
